# Mutual Fund Analytics Capstone Project

## Overview

This project analyzes mutual fund industry data using Python, SQL, and Power BI. The objective is to derive actionable insights related to fund performance, investor behavior, portfolio composition, benchmark trends, and industry growth.

## Objectives

* Analyze mutual fund industry trends
* Evaluate scheme performance using financial metrics
* Study investor demographics and transaction patterns
* Analyze portfolio holdings and sector allocations
* Compare benchmark index performance
* Build an interactive Power BI dashboard

## Dataset Modules

1. Fund Master Data
2. NAV History
3. AUM by Fund House
4. SIP Inflows
5. Category Inflows
6. Industry Folio Counts
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices

## Key Insights

* Industry AUM showed consistent growth.
* SIP inflows increased significantly across the analysis period.
* Scheme performance varied considerably across categories.
* Portfolio allocations were concentrated in a few dominant sectors.
* Benchmark indices exhibited positive long-term trends.
